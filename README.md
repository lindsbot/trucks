SF Food Truck Finder
====================

Hi! I made this app as a coding challenge, it shows a map and list view of all SF food trucks near you.

Tech Stack
----------

Node.js
Backbone.js


Data Source
-----------

Trucks](https://data.sfgov.org/Permitting/Mobile-Food-Facility-Permit/rqzj-sfat)
